# About OmniSupport

OmniSupport is a modern support ticketing website designed to streamline customer service operations. The platform connects customers, support agents, and admins in one interface for transparent ticket tracking and real-time communication.

## What the Website Does
- Allows customers to submit support requests with issue details, category, and priority.
- Enables agents to claim tickets, update ticket status, and chat with customers live.
- Gives admins a system audit view to monitor ticket progress, assignments, and support team activity.

## Who Uses It
- Customers: submit issues and follow ticket progress.
- Support Agents: view and manage assigned tickets, respond quickly, and close issues.
- Admins: audit ticket workflows, track team performance, and maintain overall system health.

## Why It Matters
OmniSupport speeds up issue resolution by making ticket status transparent and allowing support staff to collaborate instantly. It improves customer satisfaction by providing a dedicated support channel with live updates.

## Structure
- `backend/` contains the REST API, database models, authentication, and real-time socket handlers.
- `frontend/` contains the React application, UI components, and dashboard pages for each user role.

## How to Use
1. Start the backend server.
2. Start the frontend app.
3. Register or log in as a customer, agent, or admin.
4. Create tickets, claim and resolve them, and monitor progress in real time.
